# Documentação do projeto

Esta pasta organiza a evolução do **Banco de Ideias da Comunidade**.

## Como trabalhar

1. Abrir `prompts/README.md`.
2. Executar os prompts pela ordem indicada.
3. No fim de cada etapa, testar os critérios de aceitação.
4. Só avançar quando a etapa atual estiver concluída.
5. Registar decisões importantes e problemas encontrados neste diretório.

Os prompts foram divididos para evitar alterações demasiado grandes e para manter o projeto funcional durante o desenvolvimento.
