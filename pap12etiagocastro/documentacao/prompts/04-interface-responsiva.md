# Prompt 04 — Interface responsiva

## Prompt

Completa e uniformiza o CSS do projeto com base na identidade visual verde já existente. Corrige a diferença entre as classes usadas na homepage e as classes definidas no CSS. Cria estilos consistentes para cabeçalho, hero, botões, cartões, formulários, mensagens, perfil, tabelas, rodapé e estados de foco. Implementa navegação e layouts responsivos para telemóvel, tablet e computador. Não uses bibliotecas externas além da fonte já adotada, salvo necessidade justificada.

Não inventes imagens em falta: usa um bloco visual CSS ou um placeholder local claramente identificado até existir um recurso definitivo. Garante contraste, navegação por teclado, labels e estados de foco visíveis. Mantém o HTML semântico.

## Critérios de aceitação

- Não existem classes principais usadas pelo HTML sem estilo correspondente.
- A aplicação é utilizável a 320 px de largura.
- Formulários e mensagens têm apresentação consistente.
- O foco por teclado é claramente visível.
- Não existem referências a imagens inexistentes.
